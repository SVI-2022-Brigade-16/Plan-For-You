# Plan For You: сервис планирования времени

## Начало работы  

Склонировать себе репозиторий:
```
git clone https://github.com/SVI-2022-Brigade-16/Plan-For-You.git Plan-For-You
cd Plan-For-You
```

Установить NestJS CLI:
```
npm i -g @nestjs/cli
```

## Запуск

Через npm:
```
npm run start
```

Через nest:
```
nest start
```
